Primality Testing in SPARC Assembly close

Speed competition to find prime numbers in given intervals. Won 1st place with Miller-Rabin in SPARC assembly. (0.53s compared to 29.61s for 2nd place), thanks to the usage of a probabilistic primality testing algorithm (though adapted to be deterministic for the given range of numbers) and careful assembly optimization.

The assembly written took advantage of the registers available to reduce memory accesses, as is expected on a RISC architecture.

This was done in a "System Programming" course.

(Greatly simplified) clone of Mario Bros with a character that can run and jump, all in NES assembly.

This project made us experiment with the CISC architecture of a 6502-inspired CPU (CPU in the NES console) to get a good contrast with the RISC architecture of the SPARC. After discovering that this CPU lacks a mul instruction, the character's speed somehow ended up being a power of 2...

This was done as the last project of a "System Programming" course.
